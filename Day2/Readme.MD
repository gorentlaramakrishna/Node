Q, Difference between oops and object based

A, OOps : inheritance, polymorism, encapsulation.

object based : Nescape company providing objects

Node js supports modules (libraries)
predefined libraries and user defined libraries


Modules :

1, http 
    http module is used to build the http server

2, Fs (filesystem)
    Fs module is used to interact with flat files
    (textFiles, XmlFiles, json Files)

3, Express 
     express module is used to develop the rest services 
     (GET, POST, PUT, DELETE)

4, MOngoDB
     MongoDB module is used to interact with the mongodb Database 

5, MYSQl
     MySql module used to interact with mysql database 

6, mssql 
     mssql module used to interact with the SQL Server

7, multer 
    multer module used to interact with the upload and download images from the server

8, socket.io 
    socket.io is used to implement the chat appliaction

9, cors
    cors is used to interact with the port communication

    React port 3000         -      nodeport 8080

10, body-parser
     bodyparser is used to recieve the data from frontend (client application)

11, jwt
     jwt is used to implement the token based authentication

These all are modules and need to dowloand using npm or yarn
